Jose's Cheat Sheet

HTML - Table

Tag

Name

Description
<table> Table Container for all elements and information contained within the table
<thead> Table Header Display data for a specific column header within a table
<th> Table Header Data Defines a header cell in a table.
<tbody> Table Body Groups the body content in a table.
<td> Table Data Defines a cell in a table that contains data.
<tr> Table Row Display data within a row (horizontally) in a table

CSS - Positioning

Name

Description
Static Static positioning is the default that every element gets.
Relative Once the positioned element has taken its place in the normal flow, you can then modify its final position, including making it overlap other elements on the page.
Top, bottom, left, and right Used alongside position to specify exactly where to move the positioned element to.
Absolute Isolated UI features that don't interfere with the layout of other elements on the page
z-index Change the stacking order
Fixed fixes an element in place relative to the visible portion of the viewport
Sticky It allows a positioned element to act like it's relatively positioned until it's scrolled to a certain threshold
